## Type Definitions

| Type | Description |
|------|-------------|
| **voice** | Prose style inconsistency — word choice, register, sentence patterns that break the author's established voice |
| **continuity** | Factual contradiction with earlier chapters, the story bible, or the foreshadowing ledger |
| **pacing** | Scene moves too fast, too slow, or lacks appropriate rhythm for its position in the arc |
| **structure** | Chapter-level architecture — weak openings/closings, missing transitions, scene order issues |
| **prose-pattern** | Repetitive constructions, overused words, filtering verbs, adverb clusters, or other mechanical prose issues |

## Severity Definitions

| Severity | Description |
|----------|-------------|
| **CRITICAL** | Breaks the story — continuity errors, plot holes, contradicted canon decisions |
| **HIGH** | Noticeably damages reader experience — voice breaks, major pacing issues |
| **MEDIUM** | Should be fixed but doesn't break anything — structural improvements, moderate prose issues |
| **LOW** | Polish-level — minor word choices, subtle pattern improvements |
